# Java Algorithms and Data Structures
Implementations of common algorithms and abstract data structures in Java

### Full list of implemented algorithms:
* Binary Search
* Bubble Sort
* Insertion Sort
* Merge Sort
* Quick sort
  * Standard quick sort
  * Median of 3 Partitioning
  * Hybrid sort (combined with insertion sort)
  * 3 way (Dutch national flag)
* Heap Sort

### Full list of implemented data structures:
* Set (Abstract)
  * Binary search tree implementation
  * Linked list implementation
* B-Tree
* Stack (linked list)
* Priority queue (linked list)
* Weighted directed graph (adjacency list)
